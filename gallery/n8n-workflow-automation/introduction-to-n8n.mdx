## What is n8n?

n8n is an open-source workflow automation tool designed for connecting applications, APIs, and custom logic into repeatable pipelines. It functions as a programmable 'glue' between various services, offering developer-level control and the flexibility to self-host, unlike many other automation platforms [video:00:00]. It provides a robust environment for building sophisticated automated processes.
